# Pour compiler et exécuter

## Sous Linux

Nous commençons par se mettre dans le dossier src :
```bash cd src ```

Ensuite nous mettons les permissions d'éxécution pour le run.sh avec la commande suivante

```bash
chmod +x ./run.sh
```

Et enfin nous exécutons le fichier grâce à cette commande :

```bash
./run.sh
```

## Sous Windows

Ouvrez l'invite de commandes puis placez-vous dans le dossier `src` :

```bat
cd src
```

Ensuite, exécutez le fichier batch avec la commande suivante :

```bat
run.bat
```
